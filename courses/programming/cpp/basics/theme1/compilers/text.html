C++ - компилируемый язык программирования. За счёт прямой компиляции в ассемблер и оптимизаций компилятора - программы, написанные на C++, имеют высокую скорость выполнения. Несмотря на это такая компиляция имеет существенный недостаток, код на C++ в большинстве случае платформонезависем, если нет явной привязки к какой-либо платформе (ОС). Но большинство компиляторов компилируют исходный C++ код под конкретную платформу. Поэтому выбор компилятора должен осуществляться не только исходя из параметров насколько быстро происходит компиляция, как хорош он оптимизирует и какую версию С++ он поддерживает, но и компилятор должен уметь компилировать код под ту платформу, под которую вы пишите софт. 

Рассмотрим одни из самых популярных компиляторов С++ под различные платформы

  1. GCC (Linux, Android, Windows, Mac OS X)
  2. MinGW (Windows)
  3. MSVCPP (Windows)
  4. CLang (Windows, Linux, IOS, Android, MacOS)
  5. Intel C++ compiler (Linux. Windows, Mac OS X)
