Что такое аргументы функции будет рассмотрена в следующих уроках. Сейчас же стоит обратить внимание на два принимаемых функцией main аргумента: argc и argv. Через эти аргументы можно получить данные, которые могут быть переданы в качестве параметров запуска приложения. Например:

MyProgram.exe -silent -input-file data.txt -output-file result.txt

В примере при запуске программы MyProgram.exe ей передаются 5 аргументов: -silent, -input-file, data.txt, -output-file и result.txt

argc — неотрицательное значение, указывающее сколько аргументов было передано при запуске программы.

argv — указатель на массив строк. Каждый элемент массива указывает на аргументы командной строки.

Из примера:

argc = 5 argv[0] = "-silent" argv[1] = "-input-file" argv[2] = "data.txt" argv[3] = "-output-file" argv[4] = "result.txt"
