Любая программа на С++ должна содержать одну глобальную функцию main. Фунция main - это самая первая функция, которая вызывается при запуске скомпилированной программы. Про то, что такое функции - будет подробно рассказано в следующих уроках.

Функция main должна возвращать целочисленное значение типа int. Это значение называется статус кодом завершения работы программы, оно указывает, с каким статусом завершила работу программа. Если программа отработала корректно, то в таком случае функция main должна вернуть 0. Если же программа завершилась с ошибкой, то программа может вернуть любое значение, отличное от 0. 

Это может быть как отрицательное, так и положительное значение. Нет чётких стандартов, какое значение должно быть возвращаено в случае возникновении ошибки, это каждый разработчик решает для себя сам.

В зависимости от того, какое значение вернула программа - операционная система или сторонние утилиты могут его отследить и предпринять какие-либо действия, например, постараться перезапустить приложение, в случае, если оно завершило работу с ошибкой.

Допускается несколько вариантов функции main:

1. Без аргументов, с явным указанием возвращаемого типа

2. Принимающая аргументы

3. Без явного указания возвращаемого значения

Можно не указывать возвращаемое значение внутри функции main, в таком случае компилятор автоматически подставит return 0; на этапе компиляции.

Как видно из примеров, возвращаемое значение return 0; указывается внутри фигурных скобок { тело функции }. Область внутри фигурных скобок называется телом функции, внутри неё и пишется код, который должен будет быть выполнен.
